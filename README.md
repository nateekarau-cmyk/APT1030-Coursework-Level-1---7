# APT1030 Coursework Structure

This repository contains the coursework for APT1030, which is organized into levels of progressive difficulty from 1 to 7. Each level has its own directory containing relevant materials and assignments that correspond to the requirements of the coursework.

This structure allows for a gradual increase in difficulty, ensuring that students build upon their knowledge systematically, culminating in a thorough understanding and skill set related to APT1030 coursework.
