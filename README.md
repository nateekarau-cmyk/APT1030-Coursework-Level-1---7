# Repository Structure

This repository contains the coursework for APT1030, which is organized into levels of progressive difficulty from 1 to 7. Each level has its own directory containing relevant materials and assignments that correspond to the requirements of the coursework.

## Contents of the Repository:

- **Level 1:** Introduction to basic concepts and initial assignments.
- **Level 2:** Intermediate tasks that build on the foundational knowledge from Level 1.
- **Level 3:** More complex assignments that require deeper understanding and application of the concepts.
- **Level 4:** Advanced topics and projects designed to challenge the learners.
- **Level 5:** Collaborative projects that enhance teamwork skills and apply previous learnings.
- **Level 6:** Real-world scenarios and case studies for practical application.
- **Level 7:** Comprehensive tasks that synthesize knowledge from all previous levels, preparing students for future work or studies.

This structure allows for a gradual increase in difficulty, ensuring that students build upon their knowledge systematically, culminating in a thorough understanding and skill set related to APT1030 coursework.
